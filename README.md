# JSONPatcher

jsonpatcher provides a DSL to patch semantically bad JSON

## Installation

Add this line to your application's Gemfile:

    gem 'jsonpatcher'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jsonpatcher
